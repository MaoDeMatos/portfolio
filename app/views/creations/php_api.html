API PHP

API documentée sans aucun framework, écrite en pur PHP.

Projet réalisé pour la sélection de la formation de niveau 6 (Bac+3/4) Concepteur Développeur d'Applications chez Simplon.co.

Un routeur permet de séparer le site (qui ne contient que la documentation) de l'API, et donc par conséquent, les messages d'erreur en JSON des pages d'erreur HTML par exemple.
